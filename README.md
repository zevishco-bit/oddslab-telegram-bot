# 🎯 OddsLab Telegram Bot

Premium betting tips subscription bot with Telegram Stars payment integration.

## ✨ Features

✅ **Interactive 4-Question Onboarding** - Personalize user experience  
✅ **Telegram Stars Payments** - €19/month or €49/3 months  
✅ **Auto-Generated Invite Links** - One-time use per subscriber  
✅ **Daily Picks** - Automated delivery at 9 AM CET  
✅ **Metrics & Analytics** - Track conversions, payments, revenue  
✅ **Admin Dashboard** - Stats, user management, broadcasting  
✅ **User Profiles** - Database with SQLAlchemy/PostgreSQL  
✅ **Free Tips Preview** - Sample picks before payment  

## 🚀 Quick Start

### Prerequisites
- Python 3.11+
- Telegram Bot Token
- Railway account

### Setup
1. Clone repository
2. Install dependencies: `pip install -r requirements.txt`
3. Configure `.env` file
4. Run: `python bot.py`

## 🚂 Railway Deployment

1. Push to GitHub
2. Connect to Railway
3. Set environment variables
4. Add PostgreSQL database
5. Deploy!

## 📊 Admin Commands

- `/stats` - View metrics
- `/users` - List subscribers
- `/broadcast <message>` - Send announcement
